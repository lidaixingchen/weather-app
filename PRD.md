# weather-app PRD

## 核心功能

1. **城市搜索**：
   - 仅限国内城市搜索，支持拼音、中文名模糊匹配。
   - 输入框防抖（Debounce）处理。
   - 调用高德“输入提示 API”，下拉展示候选城市列表（含省份、城市、区县信息），用户选中后获取对应的 `adcode`。
2. **实时天气看板**：
   - 展示所选城市的实时温度、天气现象（晴、雨、雪等）、风向、风级、湿度。
3. **多日天气预报**：
   - 显示未来 4 天的天气预报（包含今天、明天、后天、大后天），包括气温范围（最高/最低温）、天气现象趋势。
4. **个性化生活建议**：
   - 根据当前实时温度与天气现象，在前端运行一套贴心的生活指数建议生成器。
   - 建议类型包括：穿衣指南（如防寒、清凉）、出行建议（如带伞、防晒）、运动洗车等。
5. **常用城市收藏与搜索历史**：
   - 提供“收藏城市（Pinned）”功能，可将常用城市固定在首页快捷看板。
   - 自动记录最近 5 次的搜索历史。

## 设计要求

- **新粗野主义（Neo-brutalism）设计风格**：
  - **撞色配色**：晴天使用高饱和度柠檬黄，雨天使用高对比电光蓝，雪天使用紫灰色，多云/阴天使用香草绿，整体页面呈现极具潮流感的对比色。
  - **排版与边框**：所有卡片、输入框、按钮等组件均使用 `border-4 border-black` 的粗黑线条描边，直角为主，少量倒角。
  - **平涂阴影**：卡片拥有 `8px`（按钮拥有 `4px`）的纯黑无渐变平涂硬阴影（`shadow-[8px_8px_0px_0px_#000]`）。
  - **动态气象图标**：采用具有粗黑描边特征的自定义动态 SVG 气象图标（例如太阳微旋转、雨滴上下弹跳）。
  - **微交互**：按钮与常用城市卡片在 Hover 时阴影平滑微缩，Active（点击）时产生按压位移效果（如 `translate-x-[4px] translate-y-[4px]` 且阴影微缩）。
- **响应式设计**：
  - 在移动端（Mobile）采用单列大卡片流式布局，在桌面端（Desktop）采用自适应栅格布局。
- **SEO 与性能**：
  - 符合 HTML5 语义化，添加页面 Meta 描述，支持快速的页面首屏渲染。
